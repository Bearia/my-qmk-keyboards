# pg1425_namecard

Caution: This project is BETA and not tested.

![pg1425_namecard](https://raw.githubusercontent.com/shikamiya/my-qmk-keyboards/master/kicad-files/pg1425-namecard/pg1425-namecard-ver1.png)

namecard size(55x91mm) numeric keypad with Kailh PG1425(X Switch)

Keyboard Maintainer: [shikamiya](https://github.com/shikamiya)
Hardware Supported: PG1425-namecard PCB, Pro Micro(ATmega32U4)
Hardware Availability: [PG1425-namecard PCB](https://github.com/shikamiya/my-qmk-keyboards/tree/master/kicad-files/pg1425-namecard)

Make example for this keyboard (after setting up your build environment, in qmk/qmk_firmware repository's root directory):

    ln -s /path/to/this_repository/qmk-keyboard keyboard/shikamiya
    make shikamiya/pg1425_namecard:default

See the [build environment setup](https://docs.qmk.fm/#/getting_started_build_tools) and the [make instructions](https://docs.qmk.fm/#/getting_started_make_guide) for more information. Brand new to QMK? Start with our [Complete Newbs Guide](https://docs.qmk.fm/#/newbs).
